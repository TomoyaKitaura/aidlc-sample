# Intent

## Prompt

Using AI-DLC, サンプルECサイトを構築したい。商品一覧・商品詳細・カート・注文（チェックアウト）の基本機能を持つ最小構成のECサイト。

## Summary

A greenfield, minimal sample e-commerce site to be built from scratch. The site provides the core shopping flow: browsing a product catalogue (product listing), viewing individual product details, managing a shopping cart, and placing an order through checkout. The scope is intentionally minimal — a baseline EC site covering only these four foundational capabilities, with no existing codebase to integrate with.
