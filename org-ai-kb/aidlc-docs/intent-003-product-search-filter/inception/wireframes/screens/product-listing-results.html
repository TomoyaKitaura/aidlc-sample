Wireframe annotations — State 2: Active filter with results

  • Illustrative active filter: search term e with a maximum price of 50. Results are the AND-combination of the name match and the price range (FR-4): the grid narrows from six products to the four that satisfy both conditions.
  • Results updated only on the explicit submit (Search button or Enter) — not on each keystroke (FR-5). Editing a control without submitting would leave this grid unchanged.
  • The Clear/Reset control is visible because at least one control is active (FR-8). Activating it empties all controls and restores the full catalogue.
  • Name matching is a case-insensitive contiguous substring on the name field only (FR-1, FR-2); price bounds are inclusive (FR-3). No result count is shown (Q3).
  • The echoed search term is treated as inert, output-encoded data (FR-10).
